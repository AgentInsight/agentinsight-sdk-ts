# @agentinsight-sdk/core

[![NPM](https://img.shields.io/npm/v/@agentinsight-sdk/core.svg)](https://www.npmjs.com/package/@agentinsight-sdk/core)

AgentInsight JS SDK 核心包 / AgentInsight JS SDK Core Package

共享 API 客户端、类型定义、常量、日志器和媒体处理工具。

Shared API client, type definitions, constants, logger, and media utilities.

## 安装 / Installation

```bash
npm install @agentinsight-sdk/core
```
